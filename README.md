# TagTag省エネコンテンツ FrontEnd

## Local setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Local build

Start server on `http://localhost:3000`

```bash
npm run dev
```

## AWS setup


```bash
Login to AWS console
Create Amplify project
Connect to Git repository / branch
```
